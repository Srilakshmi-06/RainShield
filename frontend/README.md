# 🛡️ RainShield - Frontend

This is the React-based frontend for **RainShield**, a parametric insurance platform for gig workers.

## 🚀 Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```
2.  **Environment Variables**:
    Create a `.env` file in this directory and add:
    ```env
    VITE_BACKEND_URL=http://localhost:5000
    ```
3.  **Run Development Server**:
    ```bash
    npm run dev
    ```
